# Simple-Sales-Prediction-

# 📊 Sales Prediction System using Machine Learning

## 🚀 Project Overview

This project is a Sales Prediction System built using Machine Learning and Flask. It analyzes sales data and predicts future sales based on input features such as quantity, price, MSRP, month, and year.

---

## 🎯 Features

* Data Analysis using Pandas & NumPy
* Data Visualization using Matplotlib & Seaborn
* Machine Learning Models:

  * Linear Regression (Baseline)
  * Random Forest (Optimized Model)
* Model Evaluation using MAE and R² Score
* Web Application using Flask
* Interactive UI with Chart.js (Line Chart Visualization)

---

## 📊 Model Performance

| Model             | MAE | R² Score |
| ----------------- | --- | -------- |
| Linear Regression | 653 | 0.76     |
| Random Forest     | 325 | 0.88     |

👉 Random Forest performed better due to its ability to capture non-linear relationships.

## ⚙️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install flask pandas numpy scikit-learn joblib matplotlib seaborn
3. Run the app:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
